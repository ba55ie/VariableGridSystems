Exponen­tial grid

The exponential grid is a graphical grid system in which the columns increase in size exponentially. You can set the amount of columns, the minimum column width and of course, the multiplier. It is meant to bring more variaty to the abundance of even width grids (most times twelve columns) and to give the designer more freedom.
